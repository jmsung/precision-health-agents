# Plan

## Goal

Build a Multi-Agent Healthcare Intelligence System for personalized and preventive medicine at the Bio x AI Hackathon (March 8, 2026).

## Approach

- Custom lightweight orchestrator coordinating specialized biomedical agents
- Anthropic Claude API as the LLM backbone
- HyenaDNA or DNABERT-2 for genomic variant interpretation
- Agents collaborate on patient data to provide evidence-grounded health guidance

## Milestones

- [ ] Core agent framework (BaseAgent, Orchestrator)
- [ ] Implement first agent end-to-end (e.g., Genomics with HyenaDNA)
- [ ] Wire up Claude API for agent reasoning
- [ ] Implement remaining agents
- [ ] Integration: multi-agent collaboration on a patient query
- [ ] Demo / presentation
